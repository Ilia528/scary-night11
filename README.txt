SCARY NIGHT — 1024 UNIQUE SKINS

1. Upload all files from this archive to the root of your GitHub Pages or Yandex Games project.
2. Keep file names unchanged.
3. Open the SKINS button in the main menu.

This build contains 1024 skins: 128 for each of 8 monsters. Skins differ by anatomy, head and eye construction, ornaments, glow/effects, animation style, and movement timing. Progress remains stored in localStorage.
